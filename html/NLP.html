Generated exploratory data analysis based on natural language processing (NLP) and text mining techniques.

View on GitHub

About the Project

As the song "Happy" by Pharrell Williams goes, "clap along if you know what happiness is to you". People could be happy for different reasons. However, do people realize what made them happy today? To figure this out, a brief study on the reasons that made people feel happy based on the dataset HappyDB was conducted. In general, this project involved some techniques of text mining, analyzing keywords and sentiments of data file. The result shows that people's happiness concentrates more achievement and affection and there is a difference between males and females as well as single and married people.

Methodology

  • Generated word cloud
  • Created pie charts and histograms to compare the frequency of happiness
  • Used network plots and bar plots to explore marital status influences
