About the Project

The purpose of this project is to implement a neural style transfer algorithm, which is able to extract the style of the style image and applying it to a content image using Convolutional Neural Networks. A pretrained VGG19 to generate content and style representations was used here. The algorithm will produce the image after keeping minimizing the total loss function. This project focused on studying the impact of different parameters on the generated image.

Methodology

  • Applied Convolutional Neural Network (CNN)
  • Constructed Loss Function (including Content Loss, Style Loss, and the Total Loss)
  • Tuned hyperparameter (Hyperparameter tuned: Random Seed, Content Layer Representation, Relative Weight, Iterations, etc.)
  • Attempted different types of style and content pictures
